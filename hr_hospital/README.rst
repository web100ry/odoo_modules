Hospital Management
===================

This module allows you to manage hospital operations efficiently.

Key Features:
-------------
* **Patients**: Maintain detailed records of patients, including personal information and medical history.
* **Doctors**: Manage doctor profiles, specialities, and their schedules.
* **Visits**: Schedule and track patient visits, record diagnoses, and provide recommendations.
* **History**: Automatically track changes in personal doctor assignments for patients.

Installation:
-------------
1. Copy the `hr_hospital` module to your Odoo addons directory.
2. Restart Odoo.
3. Update the Apps list.
4. Install the `Hospital Management` module.

Usage:
------
Access the Hospital menu from the main dashboard to start managing your hospital data.
