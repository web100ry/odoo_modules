Caritas Fleet Management System

Efficiently manage vehicle requests, trips, and fleet maintenance.

Key Features

  • Request Workflow: Streamlined process for submitting, approving, and rejecting vehicle requests.
  • Fleet Calendar: Visual representation of all vehicle trips with status-based indicators.
  • Department Management: Assign vehicles and employees to specific departments for better organization and access control.
  • Automated Notifications: Real-time alerts via Odoo Chatter and dedicated FleetBase channel for request status changes.
  • Comprehensive Reporting: Generate detailed PDF reports for vehicle usage, department activity, and personnel trips.
  • Dashboard Analytics: Dynamic charts for analyzing vehicle usage and trip distribution by department.
  • Role-Based Access: Specific permissions for Fleet Admins, Drivers, and Users to ensure data security and relevance.

User Roles

Fleet Admin

Full control over vehicles, departments, and all requests. Can approve/reject requests and access all reports and dashboards.

Fleet Driver

Assigned to specific vehicles. Can manage their trips, update trip status (Start/Complete), and view their schedule.

Fleet User

Can submit vehicle requests, track their own requests, and view vehicle availability within their department.

Technical Information

This module extends the standard Odoo Fleet application with a customized request and approval system. It integrates seamlessly with Odoo Calendar, Mail, and Reporting engines.

  • Models: vehicle.request, vehicle.trip, vehicle.department
  • Inherited Models: fleet.vehicle, res.users, calendar.event
  • Dependencies: fleet, mail, calendar, board
